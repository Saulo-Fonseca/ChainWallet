ChainWallet

This is a program designed to prevent quick access to a Bitcoin private key. If someone tries to force you to reveal your key, you won't be able to do so immediately. Instead, you'll need to input your password, base number, and exponent, which will take a significant amount of time to generate the key. This gives you time to free yourself from the situation.

This is the JavaScript version of ChainWallet. There's another 30% faster JavaScript version available at this link, but it doesn't include a progress bar or remaining time display. You won't see any changes until it's done. It's even slightly faster than the original C++ version on Github. These JavaScript versions allows people without Linux knowledge to use ChainWallet as well.

If you are recovering your key, simply follow these steps. Please enter the password for your key, along with the base and exponent, then press "Generate." This will hide your parameters for security reasons. It will also provide an estimate of how long it will take. Once the process is complete, you will receive a BIP39 mnemonic (24 words) to be used with a Bitcoin wallet such as Coinomi.

However, if you are creating it for the first time, you'll need to select a base/exponent combination that results in the desired waiting time. I recommend a minimum of 24 hours, but one week or one month would be more appropriate. Then, generate your wallet using the provided mnemonic, transfer your Bitcoins into it, and delete the wallet. Your Bitcoins are in the blockchain, not in the wallet, and the mnemonic provided here is the access to it. Be sure to preserve the public key for verification purposes. By doing so, you will only be able to recover it if you repeat the entire process. You just need to keep note of the password, base, and exponent used.

This program runs locally on your computer inside your browser. No information is sent through the internet. It contains all functions in one file. You can save this HTML and run it locally on a computer without an internet connection or in a virtual machine without a network card. This increases security and avoids unwanted access to your parameters.
